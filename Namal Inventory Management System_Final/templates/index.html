Why choose NUIMS?

Description:

The product on which we are working is NAMAL INVENTORY MANAGEMENT SYSTEM. This is a new self-contained product because in Namal there is no such type of software. This is some new stuff for Namal which will be beneficial for the current and upcoming students, faculty and staff. Main objective of making this software is that there is no such product available till now in the Namal from which the Namalites get benefited, it will be a great facility for them. The other main purpose of the product is using modern technology in order to polish our skills. This product is in the form of a website application.The user can access the website for issuing the products and will return it in time.

Sports

Namal has numerous societies for the development and healthy activities of students. Namal Sports and Adventure Society is one of the biggest societies in Namal University to conduct sports activities and thrilling adventures. There might required the stuff for the sports. So the staff will be in the namal inventory and they can issue the required items and will return it in time.

Medical

There will be medical kit in the inventory. If someone need something regarding medical will be provided. Mostly items is required for the Namal blood donation events. So if there is need of medical related stuff it will be issued if the required item is available.

ITSC

Most of the events in the Namal required some items like speakers, camera etc so it will be store in the inventory. So if someone need it for the event they will issue it and in this way it will keep track of the items issued.

Decoration

Mostly for the marketing of any event there might need the Decoration items. So they can issued decoration related items like scissors etc and can return it in time.
